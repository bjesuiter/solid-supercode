Supercode

Code: Ziel des Spiels ist es den Code mit so wenig wie möglich Versuchen zu erraten.

Regeln: Bei jedem Versuch müssen die vier Farben belegt werden. Mit einem Klick auf "Versuch prüfen" wird das Ergebnis ermittelt.
Schwarz bedeutet eine Farbe befindet sich im Code an derselben Position,
Weiß bedeutet eine Farbe ist im Code vorhanden, jedoch an einer anderen Position.

Das Spiel endet, wenn man den Code errät, man nach 7 Versuchen den Code nicht erraten hat oder aufgibt. Zum Spielende wird zudem der Code zum Vergleich angezeigt.
